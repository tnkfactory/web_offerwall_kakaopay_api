## TNK 카카오페이 혜택플러스 모바일 웹 충전소 연동 가이드 

### 1. 혜택플러스 모바일 웹 충전소 접속 URL
```
https://api2.tnkfactory.com/tnk/cpp.weboff.main?app_id={app_id}&web_yn=Y
```

| **매크로** | **설명** | **최대 길이** | **비고** |
| --- | --- | --- | --- |
| {app_id} | 매체 APP ID | string(32) | 매체별로 고정값(상수) 이다. (TNK 대시보드에서 확인 - App ID) |
